# Today Mood Logger

## 📌 프로그램 소개
Today Mood Logger는 사용자가 하루의 기분과 간단한 메모를 기록할 수 있는
콘솔 기반 파이썬 프로그램입니다.

매일의 감정을 기록함으로써 자신의 감정 변화를 돌아볼 수 있도록 돕는 것을
목적으로 기획하였습니다.

---

## 🛠 사용 기술
- Python 3
- 표준 라이브러리만 사용 (추가 설치 없음)

---

## ⚙ 실행 방법
1. Python 3이 설치되어 있는지 확인
2. 터미널에서 아래 명령어 실행

```bash
python mood_logger.py
